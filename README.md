# Data-Analysis
## About
This repository contains data analysis projects created using Python to practice real-world data cleaning, exploratory data analysis (EDA), and visualization.

## Projects
- **Sales Analysis**
  - Analyzed sales data to identify trends, top-performing products, and monthly revenue patterns.
  - Tools: Pandas, Matplotlib, Seaborn
  - Folder: `Sales_Analysis/`

## Tools & Skills
- Python
- Pandas, NumPy
- Data Cleaning & EDA
- Data Visualization
- Jupyter Notebook

## How to Run
1. Clone the repository
2. Install requirements
3. Run notebooks using Jupyter
